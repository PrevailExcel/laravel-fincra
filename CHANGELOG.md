# Changelog

All notable changes to `laravel-fincra` will be documented in this file.

## [1.0.0] - 2025-11-04

### Added
- Initial release
- Inline checkout widget integration
- Checkout redirect functionality
- Virtual accounts management
- Payouts processing
- Beneficiaries management
- Identity verification (BVN, Bank Account, IBAN, BIN)
- Balance and transaction queries
- Currency conversions
- Webhook handling with signature verification
- Comprehensive documentation
- Unit tests
- Helper functions
- Custom exceptions

### Features
- Support for multiple payment methods (Card, Bank Transfer, Mobile Money, USSD)
- Multi-currency support
- Sandbox and Live environment switching
- Automatic webhook signature verification
- Laravel 10, 11, 12+ compatibility
- PHP 7.4+ and 8.3+ support

## [Unreleased]

### Planned
- Direct Charge API integration
- More comprehensive test coverage
